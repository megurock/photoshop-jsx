# Photoshopスクリプト

過去に作ったPhotshop用JSX。   
Photshop CS5! で動作確認済み。

## 内容
- ガイドを一定間隔で作成.jsx
- レイヤーをn個複製.jsx
- レイヤーセットからスライスを作成.jsx
- 選択範囲からスライスを作成.jsx
- 選択範囲からガイドを作成.jsx


## インストール

以下のディレクトリに `megurock` をコピーしてください。  

**Mac**
/Applications/Adobe Photoshop [バージョン]/Presets/Scripts/  

**Windows**
C:\Program Files\Adobe\Adobe Photoshop [バージョン]\Presets\Scripts
